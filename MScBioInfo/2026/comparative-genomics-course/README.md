## Setup

### Install (first time)

```bash
conda env create -f environment.yml
```

### Update (after pulling new changes)

```bash
conda env update -f environment.yml --prune
```

The `--prune` flag removes packages no longer listed in `environment.yml`.

### Run

```bash
conda activate convergent-evo
cd notebooks
jupyter notebook
```

Start with `01_phylogenetics_basics.ipynb`.

### Google Colab (alternative)

If you can't install locally, open notebooks in [Google Colab](https://colab.research.google.com). Install dependencies in the first cell:

## Data

| File | Description |
|:---|:---|
| `data/cox1.nw` | 3 COX1 sequences (intro example) |
| `data/slc23a.mft.gt01` | SLC23A trimmed alignment (intro example) |
| `data/selection2_clustalo.fa` | 297 SLC26 sequences, 30 mammalian species |

| `data/species_classification.tsv` | Species, group, echolocation status |
| `data/subfamilies/` | Per-subfamily FASTAs (generated by NB00) |

All sequence headers follow the `taxid.accession` format, curated from UniProt reference proteomes.
